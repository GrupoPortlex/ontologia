HUMANAS VOLITIVAS O INTENCIONALES

Subcategoría: 1.1. Situaciones dinámicas humanas volitivas o intencionales
Definición: Describe procesos que implican dinamismo con participación humana volitiva o intencional. Dinamismo es el nodo común a la categoría.
Pregunta: ¿Se trata de una situación dinámica realizada por un humano?
Respuesta: Sí, se trata de un atropello.
Ejemplo: El atraco al banco
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso o acción de tipo médico?
Respuesta: Sí se trata de una operación planificada.
Ejemplo: El dolor de la operación
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso o acción violenta?
Respuesta: Sí, se trata de un atraco a mano armada.
Ejemplo: La muerte por disparos
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso o acción para la formación?
Respuesta: Sí, se trata de un entrenamiento para atletas de altas capacidades.
Ejemplo: El entrenamiento del atleta
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso o acción de tiempo libre?
Respuesta: Sí, se trata de una escapada de fin de semana.
Ejemplo: La escapada a la montaña
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso o acción laboral?
Respuesta: Sí, se trata del teletrabajo.
Ejemplo: Los negocios del directivo
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso cognitivo o intelectual?
Respuesta: Sí, se trata de una reflexión.
Ejemplo: El razonamiento del filósofo

MOTRICES

Subcategoría: 1.2. Situaciones dinámicas motrices
Definición: Describe procesos que implican motricidad.
Pregunta: Se trata de una situación dinámica ¿Se trata de un proceso que implica cambio motriz?
Respuesta: Sí, se trata de un bofetón de una niña mayor a una niña pequeña.
Ejemplo: El dolor del golpe/bofetón

ACONTECIMIENTOS

Subcategoría: 1.4. Situaciones dinámicas como acontecimientos
Definición: Describe procesos resultativos, causativos o no puntuales como acontecimientos.
Pregunta: Se trata de una situación dinámica ¿Se trata de un evento?
Respuesta: Sí, es un suceso.
Ejemplo: El nerviosismo del acontecimiento
Pregunta: Se trata de una situación dinámica ¿Se trata de un evento como origen?
Respuesta: Sí, es la causa.
Ejemplo: El motivo de la pelea
Pregunta: Se trata de una situación dinámica ¿Se trata de un evento resultativo?
Respuesta: Sí, surgen/se obtienen resultados.
Ejemplo: El desenlace de la situación
Pregunta: Se trata de una situación dinámica ¿Se trata de un evento definido como contratiempo?
Respuesta: Sí, surge un contratiempo.
Ejemplo: El retraso del viaje por el atasco

ENCUENTRO

Respuesta: Sí, es la exportación.
Ejemplo: La emigración a América

Ejemplo: La pregunta sobre la modernización
Pregunta: Se trata de una situación dinámica ¿Se trata de una actividad o acción de tipo económico?
